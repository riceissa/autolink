# autolink
automatically detect the title of a link

```bash
$ ./autolink.py --help
usage: autolink.py [-h] url

Get the linked title of URLs (similar to Quora and Facebook)

positional arguments:
  url         the URL

optional arguments:
  -h, --help  show this help message and exit
```
